# Xbox Battery Notifier

A small personal tool to know when the battery level of an Xbox Wireless Controller is low on Windows.

## About

This project was created for personal use to easily check the battery level of my Xbox controller, as Windows does not always provide a convenient notification when the battery is low.
Features and improvements may be added in the future depending on personal needs or user requests.

For the moment, only worked with : 
- Windows 10
- One Xbox Wireless Controller connected through Bluetooth

## Technologies

- C#
- .NET
- Windows Bluetooth GATT API
